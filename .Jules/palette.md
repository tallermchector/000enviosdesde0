## 2024-06-14 - Add ARIA Label to Navigation Menu
**Learning:** Icon-only buttons used for mobile navigation menus require descriptive `aria-label`s to be accessible to screen readers. The application's UI is in Spanish, so ARIA labels should match the language context (e.g., "Abrir menú de navegación" instead of "Open navigation menu").
**Action:** When adding accessibility attributes in this repository, always ensure they are localized to Spanish to maintain consistency with the surrounding UI.
